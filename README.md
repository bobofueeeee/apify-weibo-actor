# apify_weibo_actor
apify weibo actor for pull hot search data



# loacl develop

参考地址：https://docs.apify.com/platform/actors/development/quick-start/locally

```shell
brew install apify-cli
apify create my-actor  or apify init
cd my-actor
# main.py or main.js
apify run

# Deploy on Apify
apify login
apify push
```
